# Public Bike Rental Analysis (PostgreSQL)

## Project Overview
This project loads and processes Seoul public bike rental data
using PostgreSQL.

## Tech Stack
- PostgreSQL
- psql (\copy)
- DBeaver

## Workflow
1. Create schema
2. Create raw table
3. Load CSV using psql \copy
4. Create clean table with proper numeric casting

## Data Source
Seoul Open Data Portal – Public Bike Rental Data
